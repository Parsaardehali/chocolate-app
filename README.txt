HOW TO RUN THIS APP:

Option 1 - Double-click "Open in Chrome.bat"
   - Requires Python to be installed
   - A console window will open - keep it open while using the app
   - The app will automatically open in your browser
   - Press Ctrl+C in the console window to stop when done

Option 2 - If Python is not installed:
   1. Install Python from python.org
   2. Then use Option 1

Option 3 - Use Chrome Web Server extension:
   1. Install "Web Server for Chrome" extension from Chrome Web Store
   2. Point it to this folder
   3. Click the web server URL

Keep the console window open while using the app!
